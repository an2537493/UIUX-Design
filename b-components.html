B-End Design System

一套专为企业级电商后台打造的 UI 设计规范。

基于 Atomic Design 理念，确保设计的一致性、模块化与可维护性。

品牌识别

以 #1677FF 为核心的专业色彩体系，传递理性与信任。

组件规范

包含 30+ 基础原子组件，覆盖表单、反馈、展示等全场景。

交互原则

定义清晰的 Hover、Focus、Active 状态，确保操作反馈即时准确。

设计原则 Tip：所有的组件都遵循 4px 网格系统，界面间距请统一使用 4 的倍数 (如：4, 8, 12, 16, 24, 32px)。

Body / 14px

默认正文文本 (Body Text)。B端界面通常使用 14px 作为基础字号，行高建议设置为 1.57 (22px) 以保证长时间阅读的舒适度。如商品管理页面的 SPU 列表详情等。

Caption / 12px

辅助说明文本。用于表单提示、时间戳或次要信息。颜色通常使用 Neutral-500。

按钮 (Buttons)

用于触发操作。按钮的设计应清晰表达操作的优先级。

按钮类型 (Types)

根据重要程度选择不同的按钮类型。主按钮在每个视图中通常只有一个。

按钮尺寸 (Sizes)

适配不同的布局密度。默认为 Medium。

输入录入 (Data Entry)

基本输入框 (Basic Input)

用于单行文本输入。适用于 SPU 创建、物流发货单号录入等场景。

输入状态 (States)
